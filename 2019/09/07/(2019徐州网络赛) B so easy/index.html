传送门

题意： 就是两种操作，1：使得x点无效，2：查找大于等于x的最小有效点

解： 由于数据达到1e9不好set，所以就线段树操作就像了，节点维护区间，和存在的有效点数，然后查询的时候，用存在的有效点数剪枝一下即可。（好像有暴力过的？？？

题解用map模拟并查集简单好写，菜啊；
